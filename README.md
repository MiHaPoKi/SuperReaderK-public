Далее идет инструкция по установке и использованию.

1. Установка
   Скачайте и распакуйте файл .apk

2. Использование
   2.1. Открытие файла
        Нажмите кнопку (1) и выберите файл с расширением .epub (на данный момент это единственный поддерживаемый формат)
